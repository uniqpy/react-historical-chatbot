The client consists of;
/assets (images used for CSS/styling)
/pages (the jsx and any CSS for pages on our site that can be routed to.)
    /AboutDocs.jsx (renders an about page, showing information)
    /ChatFullPage.jsx (renders chatpage user will interact with the bot on)
/App.jsx (renders top bar of page and the help modal)
/main.jsx (renders each part of site.)

The client can be ran using "npm run dev"
